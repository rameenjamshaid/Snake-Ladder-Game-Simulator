# Snake & Ladder Game (OOP in Python)

## Project Overview
This project is a Python-based Snake & Ladder game developed using Object-Oriented Programming (OOP) concepts. It automates gameplay including dice rolling, player movement, snakes, ladders, and winner detection.

---

## Features
- Multiple players support (2–4)
- Automatic dice rolling
- Snake and ladder movement logic
- Turn-based gameplay system
- Move history tracking
- Winner declaration

---

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Modularity
- Object interaction (Player, Board, Dice, Game Controller)
